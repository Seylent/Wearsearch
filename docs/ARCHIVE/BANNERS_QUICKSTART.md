## 🚀 Як запустити

### Крок 1: Переконайтеся, що backend працює
```bash
# Має бути запущений backend на http://localhost:3001
# З підтримкою endpoints:
# GET /api/pages/home (включає банери)
# GET /api/banners
# POST /api/banners
# PUT /api/banners/:id
# DELETE /api/banners/:id
```

### Крок 2: Запустіть фронтенд
```bash
cd c:\frontend\Wearsearch-main
npm run dev
```

### Крок 3: Відкрийте головну сторінку
```
http://localhost:3000
```

Банери з'являться між Hero секцією та продуктами (якщо вони створені в адміністративній панелі).

## 🛠️ Як створити перший банер

### Варіант 1: Через UI
1. Перейдіть до `/admin`
2. Натисніть на таб **"Банери"** (іконка 🖼️)
3. Натисніть **"Додати банер"**
4. Заповніть форму:
   - **Заголовок**: "Зимова розпродаж 2026" (обов'язково)
   - **Підзаголовок**: "Знижки до 70%"
   - **Опис**: "На весь зимовий асортимент"
   - **URL зображення**: `https://images.unsplash.com/photo-1483985988355-763728e1935b?w=1920` (обов'язково)
   - **Посилання**: `/categories/winter`
   - **Текст кнопки**: "Купити зараз"
   - **Пріоритет**: 10
   - ✅ **Активний**: увімкнено
5. Натисніть **"Створити"**
6. Перейдіть на головну сторінку - банер має з'явитися!

### Варіант 2: Через API (для тестування)
```bash
# Створіть файл test-banner.json
{
  "title": "Весняна колекція 2026",
  "subtitle": "Нові надходження",
  "description": "Останні тренди сезону",
  "image_url": "https://images.unsplash.com/photo-1490481651871-ab68de25d43d?w=1920",
  "link_url": "/products/new",
  "link_text": "Дивитись зараз",
  "priority": 10,
  "is_active": true,
  "target_type": "all"
}

# Відправте POST запит (потрібен токен адміністратора)
curl -X POST http://localhost:3001/api/banners \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -d @test-banner.json
```

## 📊 Аналітика

### Автоматичне відстеження
```typescript
// Показ (impression) - коли банер з'являється
await bannerService.trackImpression(bannerId);

// Клік - коли користувач натискає
await bannerService.trackClick(bannerId);
```

### Перегляд статистики
В адмін панелі кожен банер показує:
- 👁️ Кількість показів
- 🖱️ Кількість кліків
- Співвідношення (CTR)


## 🎨 Адаптивність

### Мобільні (< 768px)
```css
h-[400px]           /* Висота банера */
text-3xl            /* Розмір заголовка */
px-4 py-2           /* Padding кнопки */
w-2 h-2             /* Розмір точок навігації */
```

### Планшети (768px - 1024px)
```css
h-[500px]           /* Висота банера */
text-4xl            /* Розмір заголовка */
px-6 py-3           /* Padding кнопки */
```

### Десктоп (> 1024px)
```css
h-[600px]           /* Висота банера */
text-6xl            /* Розмір заголовка */
px-8 py-4           /* Padding кнопки */
```
