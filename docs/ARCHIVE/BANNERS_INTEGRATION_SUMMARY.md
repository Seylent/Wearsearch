## 🔌 Backend Integration

### Endpoints використовуються

#### Публічні
- `GET /api/pages/home` - Отримання банерів на головній
- `GET /api/banners` - Список банерів

#### Адміністративні (потрібен токен)
- `POST /api/banners` - Створити
- `PUT /api/banners/:id` - Оновити
- `DELETE /api/banners/:id` - Видалити

#### Аналітика (автоматично)
- `POST /api/banners/:id/impression` - Показ
- `POST /api/banners/:id/click` - Клік

## ⚙️ Конфігурація

### Змінити час автопрокрутки
```tsx

  banners={banners} 
  autoPlayInterval={7000} // мілісекунди

```

### Налаштування таргетування
```typescript
{
  target_type: 'all',      // всі сторінки
  target_type: 'category', // конкретна категорія
  target_type: 'brand',    // конкретний бренд
  target_type: 'product',  // конкретний продукт
  target_id: 'uuid'        // ID цілі (якщо потрібно)
}
```

### Планування показу

```typescript
{

```
